# vue-knob-control
A rotary knob control for Vue.js

## Install
```shell
npm install vue-knob-control -S
```

## Usage

```javascript
import Vue from 'vue'
import VueKnobControl from 'vue-knob-control'

Vue.use(VueKnobControl)
```

## Examples

Most basic usage:
```html
<knob-control v-model="someValue"></knob-control>
```
